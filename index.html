Sobre o Projeto

O projeto Smart Garden é uma solução para monitorar e controlar o jardim e plantações automaticamente. Usando sensores e atuadores, o projeto é uma solução prática para monitorar e controlar a umidade do solo automaticamente, uma vez que, o sistema aciona a irrigação apenas quando necessário, exibindo as informações em tempo real em um display LCD. Essa abordagem otimiza o uso da água e facilita o cuidado com as plantas, garantindo o ambiente ideal para seu crescimento.

Benefícios do Smart Garden

Economia de Água

A irrigação inteligente permite economizar água, acionando a bomba apenas quando o solo está seco, ajudando a evitar o desperdício de água.

Feedback em tempo real

O display LCD mostra a umidade do solo e o status da irrigação em tempo real, permitindo que o usuário acompanhe facilmente as condições do solo.

Automação e praticidade

O sistema reduz a necessidade de monitoramento manual, permitindo irrigação automática, ideal para quem não pode supervisionar plantas constantemente.

Componentes Usados no Projeto

Conheça os principais componentes utilizados no Smart Garden para garantir o monitoramento e controle do seu jardim.

Arduino UNO R3

O Arduino Uno é uma placa microcontroladora de código aberto baseada no microcontrolador Microchip ATmega328P.

Protoboard

É uma placa de ensaio que serve como um protótipo de um aparelho eletrônico, com uma matriz de contatos.

Jumpers

Jumpers são utilizados para fazer ligações entre os componentes sem precisar fazer uso de soldas.

Leds

É um componente eletrônico semicondutor, composto de cristal semicondutor de silício ou germânio.

Resistor

O resistor é um componente que limita o fluxo da corrente elétrica em um circuito

Sensor de Umidade

Este sensor detecta os níveis de umidade no solo.

Relê

O relê é um equipamento eletrônico que permite a abertura e fechamento de um circuito elétrico.

Mini bomba d'água submersa

É um dispositivo compacto projetado para bombear pequenas quantidades de água em diversas aplicações.

Lcd

Um display de cristal líquido usado para exibir informações por via eletrônica, como texto, imagens e vídeos.

Funcionamento do Projeto

Entenda como o Smart Garden funciona para garantir o melhor ambiente para suas plantas, desde a coleta de dados até o controle automático de irrigação.

Passo 1: Leitura da umidade do solo

O Smart Garden começa coletando dados através de sensores de umidade, garantindo que cada fator ambiental seja monitorado de forma precisa.

  • - O sensor tem duas hastes que são inseridas no solo. Quanto mais água houver, melhor a condução elétrica entre as hastes, gerando um valor analógico.
  • - O Arduino lê o valor gerado pelo sensor através do pino analógico A0. Este valor representa o nível de umidade do solo. Quanto menor o valor lido, mais seco está o solo; quanto maior o valor, mais úmido.

Passo 2: Exibição das informações no display LCD

Os dados coletados são enviados para o microcontrolador (Arduino), que exibe informações em tempo real

  • - O valor da umidade lido pelo Arduino é exibido no display LCD conectado via interface I2C. O display mostra em tempo real o nível de umidade do solo.
  • - O LCD também exibe o status da bomba de água, informando se ela está irrigando ou se o solo está úmido.

Passo 3: Verificação das condições de umidade

Com base nos dados processados, o Arduino compara o valor obtido com dois limites pré-definidos no código.

  • - Se o valor lido for menor do que esse limite, o solo é considerado seco e a bomba será acionada para irrigar.
  • - Se o valor lido for maior que o limite de irrigação, mas menor que o limite de alerta, o LED amarelo indica que o solo está quase seco.
  • - Se o valor da umidade estiver acima do limite de umidade mínima, o solo é considerado úmido e a bomba permanecerá desligada.

Passo 4: Controle da bomba de água

Se o solo estiver seco (valor de umidade abaixo do limite), o Arduino ativa o relé conectado à bomba de água.

  • - O relé funciona como um interruptor. Quando o relé é acionado (estado LOW no pino de controle do relé), ele fecha o circuito e a bomba de água é ligada, bombeando água para o solo.
  • - Se o solo estiver úmido (valor de umidade acima do limite), o relé desliga a bomba, interrompendo o fornecimento de água.

Passo 5: Indicação visual com LEDs

Com base nos dados processados, o Arduino compara o valor obtido com dois limites pré-definidos no código acendo os LEDs para indicar o estado do solo

  • - LED Vermelho: Se o solo estiver seco e a bomba for acionada, o LED vermelho acende, indicando que a irrigação está em andamento.
  • - LED Amarelo (opcional): Se o solo estiver quase seco, ou seja, entre o limite de umidade para irrigação e um valor de alerta intermediário, o LED amarelo acende, indicando que o solo está perto de precisar de irrigação.
  • - LED Verde: Quando o solo estiver úmido e a bomba desligada, o LED verde acende, indicando que o solo está na condição ideal.
